# Material: 投标标书智能分析监督系统（BASS-MVP）路演

## 1. Overview (产品概述)
- **产品名称**: 投标标书智能分析监督系统 (BASS - Bid Analysis Supervision System)
- **产品定位**: 基于AI的投标文件智能分析平台，专注于发现围标、串标、陪标、关联投标等异常行为
- **核心价值**: 将传统人工审查升级为AI智能分析，监督人员审查时间减少80%
- **目标用户**: 招投标监督人员、审计人员、项目管理人员、法务人员
- **MVP版本**: 已完成P0级核心功能（标书上传、解析、多维度相似度分析、风险评分、报告生成）

## 2. Background (行业背景)
- **市场规模**: 中国公共资源交易市场规模超30万亿元，年招投标项目数超100万个
- **行业痛点**: 围标串标年涉案金额超千亿，传统的"专家评审+人工比对"模式效率低、易疏漏
- **政策驱动**: 国家发改委、财政部持续推进招投标领域"数字化监管""智能监督"
- **技术拐点**: 大模型Embedding + 国产OCR + 向量数据库技术成熟，为智能分析提供技术基础
- **时间窗口**: 2024-2026年是政企数字化监督的爆发期

## 3. Key Info (核心能力)
- **多格式解析**: 支持PDF、DOC、DOCX、ZIP，单项目批量处理≥20份标书
- **多维度分析**: 文本(30分) + 目录结构(15分) + 图片(15分) + 表格(10分) + 错别字(20分) + 元数据(10分)
- **量化评分**: 0-100分风险总分，自动划分低/中/高/严重四档风险等级
- **国产OCR**: PaddleOCR 3.0中文识别准确率>90%，支持扫描版PDF
- **向量化引擎**: Gitee AI Qwen3-Embedding-8B + ChromaDB向量数据库
- **报告输出**: 支持PDF/Word格式导出 + 在线预览，可直接作为监督证据

## 4. Evidence (应用证据)
- **典型场景**: 政府采购、工程建设、医疗设备、信息化项目等领域的招投标监管
- **效率对比**: 单项目8-10份标书的全面比对从原来的3-5天压缩到5分钟
- **检测能力**: 可识别文本雷同、模板复用、图片同源、错别字一致、元数据异常等多类串标特征
- **案例参考**: 某市政工程项目通过本系统发现3家企业文本相似度>92%，错别字一致率100%
- **国产化适配**: 全部组件支持信创环境（国产CPU、操作系统、数据库）

## 5. Analysis (深度分析)
- **市场视角**: 监督机构（发改委、审计局、财政局）+ 大型国企/央企招标方 + 第三方监管平台
- **技术视角**: Embedding + OCR + 向量检索 + 多模态比对的技术组合壁垒
- **竞争视角**: 传统人工审查 vs 单点OCR工具 vs 通用大模型问答，本产品是垂直领域的端到端解决方案
- **壁垒视角**: 业务Know-How（围标串标的判别规则）+ 行业语料 + 监督流程的深度结合

## 6. Outlook (未来规划)
- **近期 (6个月)**: 完成MVP产品化打磨，签约3-5个标杆客户（地市发改委/省审计厅）
- **中期 (1年)**: 扩展标书对比、移动端应用、大模型问答能力；开拓法务、纪检应用场景
- **远期 (2-3年)**: 构建招投标领域"监管大脑"，实现从异常检测到风险预警到案件溯源的全链路闭环
- **商业化路径**: SaaS订阅（年费50-200万）+ 私有化部署（一次性300-800万）+ 数据增值服务
